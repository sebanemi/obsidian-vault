Son registros internos especiales que el procesador usa para guardar temporalmente datos mientras realiza operaciones.

Un acumulador es como una pequeña memoria muy rápida dentro de la CPU. Se utiliza para almacenar números o resultados intermedios de cálculos. Muchas instrucciones del procesador trabajan directamente con estos registros.

Se podría pensar que A y B son dos variables "rápidas" dentro del procesador que se usan para hacer cuentas sin tener que ir a la memoria principal, que es más lenta.

Tener dos acumuladores permite:
- Operar entre ellos
- Guardar un dato mientras se usa el otro
