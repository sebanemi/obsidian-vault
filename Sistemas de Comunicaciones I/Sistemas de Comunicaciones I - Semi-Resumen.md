# 🧠 MACHETE SISTEMAS DE COMUNICACIONES I (1–17)

---

## **1. On-line / Off-line / Interactivo / No interactivo**

- **On-line**: conexión directa, tiempo real
- **Off-line**: procesamiento diferido
- **Interactivo**: hay ida y vuelta con usuario
- **No interactivo**: sin intervención del usuario

---

## **2. Interactivo en tiempo real**

- Interacción + **respuesta en tiempo acotado y predecible**
- Ej: cajero automático

---

## **3. Elementos de Internet (TCP/IP)**

- **Hosts** → envían/reciben
- **Routers** → enrutan
- **Enlaces** → transmiten

---

## **4. Niveles de estado**

- **Stateless** → sin memoria (IP)
- **Stateful** → con estado (TCP)
- **Orientado a conexión** vs **no orientado**

---

## **5. Evolución TCP/IP**

- Nace en ARPANET
- Separación TCP / IP
- 1983 estándar
- Permite crecimiento de Internet

---

## **6. Tiempo real vs diferido**

- **Tiempo real** → tiempo garantizado
- **Diferido** → sin restricción
- **Tiempo de respuesta** = ida + proceso + vuelta

---

## **7. Transmisión local**

- Área reducida (LAN)
- Alta velocidad, bajo retardo

---

## **8. DNS (dominios)**

- Jerárquico (**derecha → izquierda**)
- Ej: `pc.lab.univ.edu.ar`

---

## **9. Topología Internet**

- **Jerárquica + mallada**
- Componentes:
    - Hosts
    - Routers
    - Backbone
    - ISP

---

## **10. Proceso teleinformático**

- Campos:
    - administrativo
    - científico
    - comercial
    - social
- **Transmisión de datos**: envío emisor → receptor
- Señales:
    - analógicas
    - digitales

---

## **11. WAN empresarial**

- Tipo: WAN
- Topología: jerárquica/mallada
- Procesos: on-line + diferido
- Seguridad: firewall, VPN, auth
- Acceso: router + ISP
- Elementos: routers, switches, enlaces

---

## **14. Accesos a Internet**

- **Cableados**: fibra, ADSL, cable
- **Inalámbricos**: WiFi, 4G/5G
- **Satelital**
- **Dedicado (empresas)**

---

## **15. IETF / NIC / NOC / RFC**

- **IETF** → crea estándares
- **RFC** → documentos
- **NIC** → asigna dominios/IP
- **NOC** → opera la red

---

## **16. ISP**

- Da acceso a Internet
- Asigna IP
- Enruta tráfico
- Da servicios (DNS, mail)
- Se conecta con otros ISP

---

## **17. Internet / Intranet / Extranet**

- **Internet** → pública
- **Intranet** → privada interna
- **Extranet** → privada con acceso externo controlado

---

# 🎯 CLAVES PARA EL PARCIAL

- “Tiempo real” = **tiempo garantizado**
- DNS = **derecha → izquierda**
- Internet = **jerárquica + mallada**
- ISP = **conecta + enruta + asigna IP**
- Intranet ≠ WAN
